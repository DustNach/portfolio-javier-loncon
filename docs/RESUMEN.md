## 🔧 CÓMO USAR EL PORTAFOLIO

### Paso 1: Instalar Dependencias

Abre PowerShell o CMD en la carpeta del proyecto:

```bash
cd C:\Users\javier\CascadeProjects\portafolio-javier
npm install
```

**Si npm install falla**, prueba:
```bash
npm install --legacy-peer-deps
```

O usa Yarn:
```bash
npm install -g yarn
yarn install
```

### Paso 2: Iniciar el Servidor

```bash
npm run dev
```

El portafolio estará disponible en: **http://localhost:3000**

### Paso 3: Personalizar (Opcional)

- **Contacto**: Edita `src/components/Contact.tsx` con tu email y redes sociales
- **Proyectos**: Modifica `src/data/projects.ts` para agregar/editar proyectos
- **Información Personal**: Actualiza `src/components/Hero.tsx` y `About.tsx`

## 🆘 SOLUCIÓN DE PROBLEMAS

### Error: Cannot find module 'react'
**Solución**: Ejecuta `npm install`

### Error: Port 3000 already in use
**Solución**: Cambia el puerto en `vite.config.ts` o cierra la aplicación que usa el puerto 3000

### Errores de TypeScript en el IDE
**Solución**: Los errores desaparecerán después de ejecutar `npm install`

### npm install falla
**Solución**: 
1. Intenta `npm install --legacy-peer-deps`
2. O usa Yarn: `yarn install`
3. O instala dependencias manualmente (ver INSTRUCCIONES.md)
