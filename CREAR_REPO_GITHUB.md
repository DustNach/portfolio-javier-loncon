# 🚀 Crear Repositorio en GitHub - PASO A PASO

## 📝 Instrucciones Rápidas

### **PASO 1: Ir a GitHub**

Abre este link en tu navegador:
👉 **https://github.com/new**

---

### **PASO 2: Completar el Formulario**

Llena los siguientes campos:

1. **Repository name:** 
   ```
   portafolio-javier
   ```

2. **Description (opcional):**
   ```
   Portafolio profesional con 17 proyectos de IA, Big Data, Web, Mobile y Sistemas
   ```

3. **Visibilidad:**
   - ✅ Selecciona **Public** (para que Vercel pueda acceder gratis)

4. **Initialize this repository with:**
   - ❌ **NO marques** "Add a README file"
   - ❌ **NO marques** "Add .gitignore"
   - ❌ **NO marques** "Choose a license"
   
   (Ya tenemos todo esto en el proyecto local)

---

### **PASO 3: Crear el Repositorio**

Haz clic en el botón verde:
```
Create repository
```

---

### **PASO 4: Copiar la URL**

GitHub te mostrará una página con instrucciones. La URL será:
```
https://github.com/DustNach/portafolio-javier.git
```

---

### **PASO 5: Volver Aquí**

Una vez creado el repositorio, avísame y yo haré el push automáticamente.

---

## ⏱️ Tiempo: 1 minuto

---

## 🎯 Después de Crear el Repo

Ejecutaré estos comandos automáticamente:

```bash
git push -u origin main
```

Y tu portafolio con **17 proyectos** estará en GitHub listo para Vercel.

---

**¿Listo? Ve a https://github.com/new y crea el repositorio `portafolio-javier`**
